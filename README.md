# Student Management System (Java)

This is a console-based Student Management System developed using Java.

## Features
- Add Student
- View Students
- Search Student
- Delete Student

## Concepts Used
- Classes & Objects
- Encapsulation
- ArrayList
- Constructors
- Switch Case
- Lambda Expression

## How to Run

Compile:
javac Student.java StudentManagement.java

Run:
java StudentManagement
